== Installation ==

1. Unzip and Upload `cryptocurrency-mining-pools.zip` to the `/wp-content/plugins/` directory inside of a folder named "/cryptocurrency-mining-pools" 

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Then go to the Widgets section of your wordpress site and simply drag and drop the widget titled "Cryptocurrency Mining Pools" to any widget area of your website. You can also activate the widget through any wordpress menu on your site.
